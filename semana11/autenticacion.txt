El login es el proceso que permite identificar y autenticar a un usuario antes de darle acceso a una aplicación. 
Gracias a él, cada persona puede acceder únicamente a las funciones que le corresponden. 

Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes dentro del 
sistema. Esto ayuda a evitar accesos no autorizados y protege la seguridad de la plataforma. 

No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros 
datos sensibles de los usuarios o de la comunidad.
